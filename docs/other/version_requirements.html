Version requirements

Node.js​

Dynamode might work with older versions of Node.js, but it is recommended to use at least 16.x.x.

Dynamode versionrequirements
0.x.xNode.js >= 16.x.x

Typescript​

In order to get the most from dynamode it is recommended to use the latest Typescript version that is available.

Dynamode versionrequirements
0.x.xTypescript >= 4.x.x
