# Import

In order to use **Dynamode** you need to either `import` or `require` it in your project.

<Tabs>
<TabItem value="es-module" label="ES module" default>

```ts
import dynamode from 'dynamode';
```

</TabItem>
<TabItem value="common-js" label="CommonJS">

```ts
const dynamode = require('dynamode');
```

</TabItem>
</Tabs>

## Import specific classes/methods individually

:::info
This is recommended way of importing Dynamode.
:::

```ts
import Entity from 'dynamode/entity';
import attribute from 'dynamode/decorators';
import tableManager from 'dynamode/table';
import Dynamode from 'dynamode/dynamode';
```

## Import the whole Dynamode library

```ts
import dynamode from 'dynamode';
const { Entity, attribute, tableManager, Dynamode } = dynamode;
```

## Import specific classes/methods inside of curly brackets

```ts
import { Entity, attribute, tableManager, Dynamode } from 'dynamode';
```
