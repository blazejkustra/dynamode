# Installation

## Standard

To install Dynamode run the following command.

<Tabs>
<TabItem value="npm" label="npm" default>

```bash
npm install dynamode
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add dynamode
```

</TabItem>
<TabItem value="pnpm" label="pnpm">

```bash
pnpm add dynamode
```

</TabItem>
</Tabs>
